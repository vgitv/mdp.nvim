# mdp.nvim

Markdown presentation plugin

## Quickstart

```lua
require("mdp").setup()
```

## Start presentation

```vim
:Mdp
```

## Keymaps

| KEY  | ACTION                                     |
|------|--------------------------------------------|
| n    | Next slide                                 |
| p    | Previous slide                             |
| -    | Decrease presentation floating window size |
| +    | Increase presentation floating window size |
